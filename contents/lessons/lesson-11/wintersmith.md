# Using Wintersmtih

* [GIT]()
* [Wintersmith](wintersmith.html)
* [Parallax](parallax.html)

Wintersmith is a static site generater

## Install

	$ npm install -g wintersmith

## Run

	$ wintersmith preview

## Build

	$ wintersmith build

All files will be built to the `/build` directory
